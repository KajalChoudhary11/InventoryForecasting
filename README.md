\# Inventory Forecasting \& Optimization



\## Project Overview

This project develops a data-driven demand forecasting and inventory optimization 

system for retail supply chain operations. The goal is to reduce stockouts, 

minimize overstocking, and improve working capital efficiency.



\## Dataset

\- 1.2M+ Sales Transactions

\- 1,532 SKUs

\- 28 Store Locations

\- Weekly Inventory Snapshots



\## Methodology

\- Data Cleaning \& Preprocessing

\- Exploratory Data Analysis (EDA)

\- ABC Inventory Classification

\- Reorder Point (ROP) Calculation

\- KPI Analysis



Reorder Point Formula:

ROP = D̄ × LT + Z × σD × √LT



Where:

D̄ = Average demand  

LT = Lead time  

Z = Service level factor  

σD = Standard deviation of demand  



\## Key Results

\- Identified 42 SKUs below safety stock

\- 28 Overstocked items

\- Estimated $2.8M working capital improvement

\- 15–20% potential inventory turnover improvement
